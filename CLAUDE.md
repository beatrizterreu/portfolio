# Portfolio — Writing & Style Rules

## Content rules

- **No em dashes (—)** anywhere in the HTML content. Replace with:
  - comma `,` when parenthetical
  - colon `:` when introducing something
  - period `.` when separating two independent clauses
  - Never use `—` in titles, descriptions, alt texts, or body copy

- **Language:** all content in English

- **Role labels:** never add "(Freelance)" or similar qualifiers to role names

## Code rules

- Images in project pages: always stacked vertically (`gallery-grid--full`), never side by side
- Quotes / callouts: use `<blockquote class="project-callout">` for highlighted statements
- Spacing between `h3` and `p`: defined via `.project-body h3` in `projects.css` (`margin-bottom: var(--space-md)`)
